This package contains two ready-to-paste Claude Code prompts.

1) CLAUDE_CODE_PREDEPLOY_PROMPT.md
   - Runs a predeployment readiness audit (read-only) and produces deployment instructions.

2) CLAUDE_CODE_APP_REG_PROMPT.md
   - Produces a separate runbook for Entra app registrations and cross-tenant Teams setup (instructions only).

Copy these into your Claude Code session(s).
